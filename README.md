# TaskTeam

TaskTeam — это современная веб-платформа для управления задачами в командах. Система позволяет создавать проекты, назначать задачи, отслеживать прогресс и эффективно организовывать работу команды.

## 🚀 Функции

- **Управление пользователями**: Регистрация, аутентификация и авторизация
- **Командная работа**: Создание команд и управление членами команды
- **Проекты**: Организация задач в проекты с цветовой кодировкой
- **Задачи**: Создание, редактирование и отслеживание статуса задач
- **Приоритеты**: Установка приоритетов задач (Низкий, Средний, Высокий, Критический)
- **Статусы задач**: Отслеживание прогресса (TODO, В работе, На проверке, Готово, Просрочено)
- **Современный интерфейс**: Адаптивный веб-интерфейс с градиентным дизайном

## 🛠 Технологии

### Backend
- **FastAPI** — высокопроизводительный веб-фреймворк для Python
- **SQLAlchemy** — ORM для работы с базой данных
- **SQLite** — база данных (можно заменить на PostgreSQL/MySQL)
- **JWT** — аутентификация с использованием JSON Web Tokens
- **Pydantic** — валидация данных

### Frontend
- **HTML5/CSS3** — современный веб-интерфейс
- **JavaScript** — интерактивность (ES6+)
- **Fetch API** — асинхронные запросы к API

### Инфраструктура
- **Docker** — контейнеризация приложения
- **Docker Compose** — оркестрация сервисов
- **Nginx** — веб-сервер и reverse proxy

## 📋 Требования

- Docker и Docker Compose
- Python 3.8+ (для локальной разработки)

## 🚀 Установка и запуск

### С помощью Docker Compose (рекомендуется)

1. Клонируйте репозиторий:
```bash
git clone https://github.com/your-username/taskteam.git
cd taskteam
```

2. Создайте файл `.env` с переменными окружения (опционально):
```bash
SECRET_KEY=your-super-secret-key-here
```

3. Запустите приложение:
```bash
docker-compose up --build
```

4. Откройте браузер и перейдите по адресу: `http://localhost:8080`

### Локальная разработка

1. Клонируйте репозиторий и перейдите в директорию проекта

2. Создайте виртуальное окружение:
```bash
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# или
.venv\Scripts\activate     # Windows
```

3. Установите зависимости:
```bash
pip install -r backend/requirements.txt
```

4. Запустите базу данных (SQLite будет создана автоматически)

5. Запустите сервер разработки:
```bash
cd backend
python -m uvicorn main:app --reload
```

6. В другом терминале запустите Nginx для фронтенда или откройте `frontend/index.html` напрямую

## 🔧 Конфигурация

### Переменные окружения

- `DATABASE_URL` — URL базы данных (по умолчанию SQLite)
- `SECRET_KEY` — секретный ключ для JWT токенов
- `ALGORITHM` — алгоритм шифрования JWT (по умолчанию HS256)
- `ACCESS_TOKEN_EXPIRE_DAYS` — срок действия токена в днях

### База данных

По умолчанию используется SQLite. Для продакшена рекомендуется PostgreSQL или MySQL.
